
ABRA O PROMPT DE COMANDO E EXECUTE:
npm install -g create-react-app

--------------------------------------

DEPOIS DA RESPOSTA DESSE PRIMEIRO COMANDO ACIMA, EXECUTE:

create-react-app nome-do-app


-------------------------------------

DEPOIS, SE TIVER USANDO O YARN, EXECUTE:

yarn start

Geralmente, o comando rendereiza a aplicação criada em localhost:300
